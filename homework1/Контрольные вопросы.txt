1. Основные характеристики
-Компилируемый 
Для того, чтобы запустить программу, её нужно сначала скомпилировать
-Низкоуровневый
Позволяет работать напрямую с ресурсами компьютера
-Процедурный 
Обрабатывает данные с помощью алгоритмов
-Объектно-ориентированное программирование
Делает акцент на данных, сокращает код и создаёт классы
2. Фундаментальные типы данных
Логический тип bool
Символьный тип char
Целые типы int
Типы с плавающей точкой double
Перечисленные типы для представления значений из конкретного множества enum
Тип используемый для указания отсутствия информации void

4. Определение - определение сущности (реализация типа), которая 
соответствует идентификатору или выделение памяти объекту
Объявление - введение идентификатора(имени) + описание типа
Инициализация - определение начального состояния объекта
Присваивание - операция присвоения переменной некоторого значения
